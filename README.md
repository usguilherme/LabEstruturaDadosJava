# Estrutura de Dados em Java

Implementações de estruturas de dados e algoritmos clássicos desenvolvidas durante meus estudos de **Ciência da Computação** na **UFCG**.

Este repositório contém implementações próprias das principais estruturas ensinadas em disciplinas de **Estrutura de Dados e Algoritmos**, utilizando **Java**.

---

## Estruturas Implementadas

* Binary Search Tree (BST)
* Binary Heap
* Linked List
* Queue
* Stack
* Binary Search

---

## Algoritmos de Ordenação

* Bubble Sort
* Selection Sort
* Insertion Sort
* Merge Sort
* Quick Sort
